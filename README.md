A complete collection of front-end web development assignments completed as part of the **Ethnus MERN Full Stack Developer Internship Program**, covering HTML, CSS, Bootstrap, and JavaScript fundamentals through hands-on practical tasks.

## 📌 Overview

This repository contains two assignments submitted during the Ethnus MERN Full Stack certification program. Each assignment progressively builds on core web development concepts — from basic HTML structure to styled components, responsive Bootstrap layouts, and interactive JavaScript functionality.

## 📚 Assignment Breakdown

### 📝 Assignment 1 — HTML, CSS & Bootstrap Fundamentals

**HTML Tasks (5 Tasks)**
- Structured data tables with custom styling
- HTML forms with input validation
- Semantic HTML elements and document structure
- Lists, links, and media embedding

**CSS Tasks (7 Tasks)**
- Styled tables with border-radius and color themes
- CSS Grid layout with geometric shapes (rectangles, ovals)
- Flexbox-based responsive layouts
- CSS animations and transitions
- Advanced selectors and pseudo-classes

**Bootstrap Tasks (6 Tasks)**
- Responsive grid system with breakpoints
- Bootstrap navigation bars and dropdowns
- Cards, modals, and UI components
- Bootstrap form components and validation styles
- Utility classes for spacing, typography, and colors

---

### 📝 Assignment 2 — Advanced Front-End & JavaScript

**Advanced HTML Tasks (5 Tasks)**
- Complex form structures
- Semantic HTML5 elements
- Accessible markup patterns

**Advanced CSS Tasks (7 Tasks)**
- Multi-column layouts
- CSS custom properties (variables)
- Advanced responsive design patterns

**Advanced Bootstrap Tasks (6 Tasks)**
- Complex component combinations
- Custom Bootstrap theming
- Responsive multi-section layouts

**JavaScript Tasks (9 Tasks)**
- DOM manipulation and event handling
- Form validation with JavaScript
- Dynamic content rendering
- User registration and display system (`register.html` + `users.html`)
- Interactive UI components
